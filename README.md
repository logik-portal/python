# Logik Portal Python Scripts

This repository is a collection of scripts contributed by members of the Autodesk Flame/Logik community.  We welcome contributions from everyone!

1. **Using the Logik Portal (Recommended):** Download the scripts directly from this repository using the Logik Portal within Flame. This automatically installs them in the correct location.
